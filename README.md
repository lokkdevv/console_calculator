# Console Calculator
Simple console calculator made with C++
# Features
+ Basic arithmetic operations: addition, subtraction, multiplication, division.  
+ Support for parentheses.  
+ Error handling for invalid inputs.
# Usage
+ Run the application by double clicking the exe or running ```./main.exe``` in the console, then write your mathematical expression in the console and press enter, for example:
  ```
  ~9*9
  81
  ~(7+7)*2
  28
  ```
